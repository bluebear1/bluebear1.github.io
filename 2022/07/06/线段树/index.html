##结构解析
线段树的目的是实现在logN时间内动态修改区间值和修改值,主要逻辑是把区间[1，N]划分成logN个层级，[2N,4N)个区间

  • 区间数据结构:每个区间这里由Node表示，不记录区间l,r，由[1,N] 二分获得，left,right分别代表左右二分子区间，
    val代表区间和，add代表子区间的每个树都需要+add，add大概是这个区间每个数都*了多少倍;
  • query:在[sl,sr]内查询 [l,r]范围内的区间和
  • update:在[sl,sr]内将[l,r]区间内每个元素+add
  • pushDown:需要切分父区间的时候（二分切分），动态创建子区间，并且根据区间长度和add设置好子区间val,add
  • pushUp:主要是用于更新的时候，[sl,sr]被切分成2N个子区间，每个子区间值计算完后，递归求和logN次

##使用场景
区间修改求和，区间最大值，最小值，最大公因数等等

###Range模块
覆盖区间或清理
Range模块是跟踪数字范围的模块。设计一个数据结构来跟踪表示为 半开区间 的范围并查询它们。
半开区间 [left, right) 表示所有 left <= x < right 的实数 x
实现 RangeModule 类:
RangeModule() 初始化数据结构的对象。
void addRange(int left, int right) 添加 半开区间 [left, right)，跟踪该区间中的每个实数。添加与当前跟踪的数字部分重叠的区间时，应当添加在区间 [left, right]中尚未跟踪的任何数字到该区间中。
boolean queryRange(int left, int right) 只有在当前正在跟踪区间 [left, right) 中的每一个实数时，才返回 true ，否则返回 false。
void removeRange(int left, int right) 停止跟踪 半开区间 [left, right) 中当前正在跟踪的每个实数
